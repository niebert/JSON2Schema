JSON Editor Examples

JSON to Schema Generator json2schema.html

by Engelbert Niehaus

These examples use the JSON Editor of Jeremy Dorn to create a JSON Schema. Furthermore Font-AweSome and the ACE-Editor are integrated in the examples for offline use of the JSON-Editor. The very helpful examples provided by Jeremy Dorn must be altered, so that CSS layout and required libraries are available on the local device for offline application of the JSON Editor.

The current examples can be found in docs/-folder of the git repository json-editor (fork Jeremy Dorns JSON Editor>. The examples demonstrate the following:

  • You can edit a raw JSON in the textarea and submit the JSON to json2schema.html. By creating a HTML form in your own web-site you can populate the JSON2Schema editor with a JSON of your choice remotely (see wiki page of JSON editor)

  • json2schema.html example provides an example, how you can create a JSON schema from a given JSON file. A JSON Schema is needed to feed the JSON editor with the structure of the desired JSON output delivered by Jeremy Dorns JSON-Editor.
